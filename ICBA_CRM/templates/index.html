A basic CRM software for ICBA

A web hosting service allows individuals and organizations to make their website accessible via the World Wide Web.

Our Project

Welcome to our Customer Relationship Management website allow users to keep tract of individuals and organizations that are in contact with ICBA.

Our basic CRM allows tracking of individuals and organizations, users are able to create new entries, read the information available for each entry, modify and save the existing information, and remove old and unnecessary entries from the book. Additionally, users can import data from csv and export to a file format of their choice.



Built with ease

This website was built using the 'Django' web framework with 'Bootstrap' for the front end and 'SQLite' for the database.

BOOTSTRAP

Built with the latest version of Bootstrap, Bootstrap 4.

DJANGO

Built with the latest version of Django, Django 3.0.7.

SQLite

Used SQLite3 as a relational database to store data .
